# 画风新增海报/公告/电影宣传 — 设计规格

**日期：** 2026-05-24  
**状态：** 已批准

## 目标

在现有画风选项中新增"海报风格"、"公告风格"、"电影宣传风格"三种应用场景选项，用户选择后提示词自动融入对应风格描述。

## 用户故事

1. 用户在步骤 2（画面设置）选择画风时，看到新增的三个选项
2. 用户选择"海报风格"后，生成的提示词自动包含"竖版构图，视觉冲击力强"等描述
3. 用户不选择任何新风格时，按原有逻辑生成提示词（插画、动漫等）
4. 画幅方向（竖屏/横屏）与风格独立，用户自由选择

## 设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 新增字段 | 否，复用 `artStyle` | 避免过度设计，保持类型系统简洁 |
| UI 形式 | 标签选择器，不分组 | 与现有画风交互一致，改动最小 |
| 提示词生成 | 自动融入风格描述 | 新选项本身包含描述文本，自然嵌入 |
| AI 推荐 | 不区分风格 | 保持现有热门推荐逻辑不变 |
| 画幅方向 | 不强制 | 三种风格都不绑定画幅 |

## 改动范围

| 文件 | 改动内容 |
|------|----------|
| `src/types/index.ts` | 无改动 |
| `src/components/SettingsStep.tsx` | `ART_STYLE_OPTIONS` 数组新增 3 个选项 |
| `src/app/api/prompt/route.ts` | 无改动 |
| `src/app/api/style/route.ts` | 无改动 |
| `src/app/api/hot-options/route.ts` | 无改动 |

## 新增画风选项

```
海报风格，竖版构图，视觉冲击力强，适合宣传展示
公告风格，横版横幅，包含标题文字区域，信息清晰
电影宣传风格，电影级光影，叙事感强，适合剧照海报
```

## 向后兼容

- 不选新风格 → `artStyle` 为空 → 按原逻辑生成
- 选新风格 → `artStyle` 包含描述文本 → 提示词自动融入
- 现有用户数据不受影响
